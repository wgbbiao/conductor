## Global Constraints

- 复用 P1 的全部 Global Constraints（Node ≥20.10、pnpm ≥9、TS strict 禁 any、PG 唯一事实源、Conventional Commits、中文注释英文标识符）
- **密码绝不落明文**：bcrypt hash（cost ≥12），hash 不进 AuditEvent/payload/日志
- **JWT secret 不进代码**：从 `JWT_SECRET` 环境变量读，`.env.example` 提供占位
- **User 表按多用户建模**：即使 P1.5 只有 1 个用户，字段（id/email/passwordHash/role/...）按多人设计，不为单用户简化
- **不做**：注册界面、OAuth/SSO、邮箱验证、找回密码、用户列表 CRUD UI、完整 RBAC 矩阵
- 鉴权守卫失败统一返回 `401 Unauthorized`，不泄露"用户存在与否"

- [ ] **Step 2: `auth.service.ts`（TDD：先写测试）**

> 先写失败测试 `auth.service.spec.ts`：mock PrismaService，测（a）正确密码→返回 token+user（不含 hash）；（b）错密码→抛 Unauthorized；（c）用户不存在→抛 Unauthorized（与错密码同消息，防枚举）。

```ts
import { Injectable, UnauthorizedException } from "@nestjs/common";
import { JwtService } from "./jwt.service";   // Task 5 提供；此处先注入接口
import * as bcrypt from "bcrypt";
import type { User } from "@conductor/core";

@Injectable()
export class AuthService {
  constructor(
    private prisma: PrismaService,            // 同 P1
    private jwt: JwtService,
  ) {}

  async login(email: string, password: string): Promise<{ token: string; user: Omit<User, never> }> {
    const user = await this.prisma.user.findUnique({ where: { email } });
    // 用户不存在与密码错返回同一消息，防枚举
    const ok = user ? await bcrypt.compare(password, user.passwordHash) : false;
    if (!user || !ok) throw new UnauthorizedException("凭据无效");

    const token = await this.jwt.sign({ sub: user.id, role: user.role });
    // 返回的 user 不含 passwordHash
    const { passwordHash, ...safe } = user;
    return { token, user: safe };
  }

  /** 校验 JWT payload，返回 AuthContext（守卫用） */
  async validatePayload(payload: { sub: string; role: string }) {
    const user = await this.prisma.user.findUnique({ where: { id: payload.sub } });
    if (!user) return null;
    return { userId: user.id, role: user.role };
  }
}
```

## Task 5: JWT 基础设施 + 全局守卫

**Files:**
- Create: `apps/api/src/auth/jwt.service.ts`, `auth/jwt-auth.guard.ts`, `auth/auth.module.ts`

- [ ] **Step 1: `jwt.service.ts`（薄封装 jsonwebtoken）**

```ts
import { Injectable } from "@nestjs/common";
import * as jwt from "jsonwebtoken";
import { config } from "../config";

@Injectable()
export class JwtService {
  sign(payload: { sub: string; role: string }): string {
    if (!config.jwtSecret) throw new Error("JWT_SECRET 未配置，拒绝启动");
    return jwt.sign(payload, config.jwtSecret, { expiresIn: config.jwtExpiresIn as any });
  }
  verify(token: string): { sub: string; role: string } {
    return jwt.verify(token, config.jwtSecret) as { sub: string; role: string };
  }
}
```

- [ ] **Step 2: `jwt-auth.guard.ts`（全局守卫）**

```ts
import { CanActivate, ExecutionContext, Injectable, UnauthorizedException } from "@nestjs/common";
import { JwtService } from "./jwt.service";
import { AuthService } from "./auth.service";
import type { AuthContext } from "@conductor/core";

@Injectable()
export class JwtAuthGuard implements CanActivate {
  constructor(private jwt: JwtService, private auth: AuthService) {}

  async canActivate(ctx: ExecutionContext): Promise<boolean> {
    const req = ctx.switchToHttp().getRequest();
    const header = req.headers.authorization ?? "";
    const match = /^Bearer (.+)$/.exec(header);
    if (!match) throw new UnauthorizedException("缺少 Bearer token");
    try {
      const payload = this.jwt.verify(match[1]!);
      const authCtx: AuthContext | null = await this.auth.validatePayload(payload);
      if (!authCtx) throw new UnauthorizedException();
      req.user = authCtx;   // 注入到请求，controller 可取
      return true;
    } catch {
      throw new UnauthorizedException("token 无效或已过期");
    }
  }
}
```
